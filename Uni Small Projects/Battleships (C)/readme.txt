======================= Instrukcja Kompilacji ================
Program można skompilować po otwarciu załączonego projektu 
w edytorze codeblocks. Program został napisany z myślą o
systemie windows.

======================= Opis programu =========================
Załączony program jest implementacją tradycyjnej gry w statki
dla dwóch graczy. Gra jest obsługiwana w trybie tekstowym.
Dodatkowe instrukcje są dostępne w menu gry po wywołaniu ekranu
pomocy.
